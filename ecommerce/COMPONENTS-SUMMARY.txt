================================================================================
EVENT HANDLERS
================================================================================

All Components Follow Pattern:
1. Handler defined as function
2. Check if handler != none
3. Validate state (inStock, etc.)
4. Call with appropriate data
5. Parent receives event + product data

Example Pattern:
```
handleAddToCart = fn() {
    if onAddToCart != none && inStock {
        onAddToCart()
    }
}
```

Integrated in Grid:
```
onAddToCart={() => handleAddToCart(product)}
```

Allows parent to receive:
```
onAddToCart={fn(product) { addToCart(product) }}
```
